Установка и запуск:\
– Скачайте или склонируйте репозиторий к себе на копьютер.\
– Напишите yarn или npm install в терминале (установка зависимостей).\
– Запустите приложение (yarn start или npm run start).\
\
Список зависимостей:\
cors: ^2.8.5\
dotenv: ^16.0.3\
express: ^4.18.1\
mongoose: ^6.6.1\
nodemon: ^2.0.20
